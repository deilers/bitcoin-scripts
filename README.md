# Bitcoin Scripts
Convenience scripts used to extract information from bitcoin-cli outputs.

## Requirements
- Bash terminal
- Node.js + NPM
- [Bitcoin Core full node](https://bitcoincore.org/en/download/)

## How To Use
- Install yarn
  - ``` npm i -g yarn ```
- Install dependencies
  - ``` yarn install ```
- Make Scripts executable
  - ``` chmod +x <scriptname>.sh ```
